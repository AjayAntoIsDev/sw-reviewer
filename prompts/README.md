# Prompt Pack

This folder contains a modular Shipwright reviewer prompt set derived from the provided guidelines.

Files:

- `00_system_prompt.md`: base role and decision philosophy
- `01_review_workflow.md`: default review process
- `02_general_rules.md`: global rules and proof video requirements
- `03_project_type_checks.md`: category-specific shipping requirements
- `04_auth_and_testing.md`: auth and functional testing standards
- `05_readme_review_prompt.md`: README evaluation criteria
- `06_decision_output_prompt.md`: required response structure
- `07_response_templates.md`: ready-to-use approval and rejection templates

Suggested usage:

- Use `00_system_prompt.md` as the base system prompt.
- Layer `01` through `06` as review instructions.
- Use `07_response_templates.md` for consistent reviewer output.
